# Queue Cure '26

## Overview

Queue Cure is a real-time clinic queue management system designed to reduce patient waiting uncertainty and simplify queue handling for receptionists.

## Problem Statement

Many clinics still rely on manual token systems, causing long waiting times and lack of visibility for patients. Queue Cure provides a digital solution with live queue updates.

## Features

### Receptionist View

* Add new patients
* Generate and manage tokens
* Call the next patient
* Set average consultation time

### Patient View

* View the current token being served
* Check the number of tokens ahead
* See estimated waiting time

### Real-Time Updates

* Both screens update instantly when the receptionist calls the next token.

## Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB
* Real-Time Communication: Socket.IO

## Installation

1. Clone the repository
2. Install dependencies:
   npm install
3. Start the backend server:
   npm run server
4. Start the frontend:
   npm start
